  1. ¿Que es un sistema de control de versiones?

    es una herramienta de software esencial en el desarrollo de software que registra los cambios en archivos a lo largo del tiempo

  2. ¿Cual es la diferencia entre git y github?

    Git es un sistema de software local para controlar las versiones y el historial de cambios en el código, mientras que GitHub es una plataforma en la nube (sitio web) que aloja repositorios de Git para facilitar el trabajo colaborativo, la copia de seguridad y la gestión de proyectos en línea

  3. ¿Cómo creo una rama en git?

    con el comando git checkout -b nombre-rama

  4. ¿Cómo clono un repositorio?

    ejecutando git clone "URL" siendo esta la del repositorio

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    con el comando git pull

  6. ¿Cómo puedo generar un tag?

    usando git tag v1.0 -m "Versión 1"

  7. Cómo hago una mezcla de ramas?

    con el comando git merge

  8. ¿Cómo puedo moverme a un commit especifico?

    con el comando git checkout
